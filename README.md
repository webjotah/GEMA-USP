# GEMA - Grupo de Estudos em Algoritmos

## What is GEMA?

GEMA is a study group affiliated with ICMC (Instituto de Ciências Matemáticas e de Computação) at USP (Universidade de São Paulo). The group focuses on learning and practicing algorithms and data structures through solving programming problems, typically from online judges like URI Online Judge.

## What's in this repository

This repository contains C++ solutions to various programming problems organized by weeks. Each week folder includes:

- `.cpp` files: Source code solutions to individual problems
- `output/` folder: Likely containing sample outputs or test results

The problems cover a range of topics including basic arithmetic, data structures, algorithms, and more advanced concepts as the weeks progress.

### Structure:
- **week1/**: Introductory problems (e.g., bananas.cpp, corrida.cpp)
- **week2/**: Intermediate problems (e.g., amir.cpp, bidirecional.cpp, consecutivos.cpp, etc.)
- **week3/**: Advanced problems (e.g., acoes.cpp, codigo.cpp, hanoi.cpp, etc.)
- **week4/**: Complex algorithmic challenges (e.g., concert.cpp, fib.cpp, tlights.cpp, etc.)

# GEMA - Grupo de Estudos em Algoritmos

## O que é GEMA?

GEMA é um grupo de estudos afiliado ao ICMC (Instituto de Ciências Matemáticas e de Computação) da USP (Universidade de São Paulo). O grupo se concentra em aprender e praticar algoritmos e estruturas de dados por meio da resolução de problemas de programação, tipicamente de juízes online como o URI Online Judge.

## O que há neste repositório

Este repositório contém soluções em C++ para vários problemas de programação organizados por semanas. Cada pasta de semana inclui:

- Arquivos `.cpp`: Código fonte das soluções para problemas individuais
- Pasta `output/`: Saídas de exemplo ou resultados de testes (basta ter o gcc instalado e rodar o comando g++ (arquivo).cpp -o output && ./output)

Os problemas cobrem uma variedade de tópicos incluindo aritmética básica, estruturas de dados, algoritmos e conceitos mais avançados conforme as semanas avançam.

### Estrutura:
- **week1/**: Problemas introdutórios (ex.: bananas.cpp, corrida.cpp)
- **week2/**: Problemas intermediários (ex.: amir.cpp, bidirecional.cpp, consecutivos.cpp, etc.)
- **week3/**: Problemas avançados (ex.: acoes.cpp, codigo.cpp, hanoi.cpp, etc.)
- **week4/**: Desafios algorítmicos complexos (ex.: concert.cpp, fib.cpp, tlights.cpp, etc.)
